# Client

React frontend for the poetic storytelling platform.

## Tech

- React + JSX
- React Router
- Axios
- Framer Motion
- Pure CSS

## Run

```bash
cd client
npm install
cp .env.example .env
npm run dev
```

Admin dashboard:

```text
http://localhost:5173/v1/adminShreyaTiwari
```

## Build

```bash
npm run build
npm run preview
```

## Environment

```env
VITE_API_BASE_URL=http://localhost:5000/api
```
